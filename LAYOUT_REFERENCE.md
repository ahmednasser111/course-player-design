## Responsive Breakpoints

```typescript
// Mobile First Approach
// < 640px: Mobile (sm)
// 640px - 1024px: Tablet (md, lg)
// > 1024px: Desktop (xl)

// Key Adjustments:
// - Controls: Hide labels on mobile, show on desktop
// - Sidebar: Hidden on mobile, visible on desktop wide mode
// - Modals: Full screen on mobile, centered on desktop
// - Grid: 1 column on mobile, 3+ columns on desktop
```

## Animation & Transitions

```
Smooth Scroll:
- behavior: 'smooth'
- duration: ~300ms

Player Mode Transitions:
- Duration: ~200ms ease-in-out
- Z-index changes for proper layering

Hover Effects:
- Button background: 100ms transition
- Text color: 100ms transition
- Shadow: 150ms transition

Modal Animations:
- Open: ~150ms fade-in scale
- Close: ~150ms fade-out scale
- Backdrop: ~200ms fade
```
